# Gerenciador de Biblioteca Pessoal

## Aluno: Luis Bernardo Pessanha Batista

## Descrição

Aplicação em TypeScript para gerenciar uma biblioteca pessoal via terminal.

## Funcionalidades

- Cadastro e remoção de livros
- Busca por título
- Filtro por autor
- Controle de leitura
- Estatísticas
- Classificação por década

## Como executar

```bash
npm install
npm run dev
node dist/index.js
```
